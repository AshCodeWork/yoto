    Menu组件

    项目内嵌了一个针对element-ui纵向菜单组件的封装。可以针对传入的数据自动区分使用<el-submenu /> 和 <el-menu-item />

    组件在 src/views/home/components/Menu.vue中

    使用

    属性说明：

    属性名 功能
    title 标题
    icon 栏目对应的图表（使用element-u自带的）
    disabled 如果有并且是true，则不能使用
    link 跳转地址
    children 如果有，则表示还有下一层级；如果没有下一层级，建议不用使用children
