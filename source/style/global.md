# global样式

## 浮动部分

|class名| 功能 |
|:---|:---|
| fr | 左浮动 |
| fl | 右浮动 |
| clearfix | 清除浮动 |

## 显示部分

|class名| 功能 |
|:---|:---|
| hide | 隐藏 |
| show | 显示 |
| visible | overflow恢复默认值 |

## 文本部分

|class名| 功能 |
|:---|:---|
| text-overflow | 超过一行... |
| text-overflow-2 | 超过二行... |
| text-left | 文本左对齐 |
| text-right | 文本右对齐 |
| text-center | 文本居中对齐 |
| fbold | 文本加粗 |
| fdel | 文本删除线 |


## 鼠标部分

|class名| 功能 |
|:---|:---|
| cursor | 让按钮变成小手 |
| unselect | 光标无法选中 |
